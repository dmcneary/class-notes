## Sample Quiz

1. Let $A = \{v, w, x, y, z\}$. Which of the following statements are true:
>- $\varnothing\sub A \times A$ : **TRUE**
>- $\varnothing\in A$ : **FALSE**
>- $\varnothing\sube A \times A$ : **TRUE**
>- $\varnothing\sube A$ : **TRUE**

2. Let $A = \{v, w, x, y, z\}$. Which of the following statements are true:
>- $\{x, y\}\in P(A)\times P(A)$ : **FALSE**
>- $(x, y)\in P(A)\times P(A)$ : **FALSE**
>- $\{x\}, \{y\}\in P(A)\times P(A)$ : **FALSE**
>- $(\{y\}, \{x\})\in P(A)\times P(A)$ : **TRUE**

3. $U = \{1, 2, 3, 4, 5\}$ is the universal set. $A = \{1, 5\}, B = \{3, 4, 2\}, C = \{2, 5\}$. List all elements of $(B \cup A) - (C \cap A)$, in ascending order.
>$B \cup A = \{1, 2, 3, 4, 5\}$
>$C \cap A = \{5\}$
>$(B \cup A) - (C \cap A) = \{1, 2, 3, 4, 5\} - \{5\} = \{1, 2, 3, 4\}$

4. List the elements of the set $\{t \in Z \vert t^2 \le 10\}$ in ascending order.
>$\{-3, -2, -1, 0, 1, 2, 3\}$

5. $U = \{1, 2, 3, 4, 7, 8, 9, 10\}$ is the universe, $A = \{2, 3, 8, 9\}, B = \{1, 7, 10\}, C = \{2, 4, 10\}$. List all the elements of $\overline{((B\cap C)\cup A)}$.
>$ = \overline{(B\cap C)} \cap \overline{A}= (\overline{B}\cup\overline{C}) \cap \overline{A}$
>$ = (\{2, 3, 4, 8, 9\}\cup\{1, 3, 7, 8, 9\})\cap\{1, 4, 7, 10\}$
>$ = \{1, 4, 7\}$

6. $U = \{1, 2, 3, 4\}$ in the universal set. $A = \{2, 4\}, B = \{2, 3\}, C = \{2, 3, 4\}$. Which of the following is $B \times C$:

>- $\{(2, 3), (2, 4), (3, 2), (3, 4)\}$ : **FALSE**
>- $(2, 2), (2, 3), (2, 4), (3, 2), (3, 3), (3, 4)$ : **FALSE**
>- $\{(2, 2), (2, 3), (2, 4), (3, 2), (3, 3), (3, 4)\}$  : **TRUE**
>- $\{(2, 2), (3, 3), (4, 4), (2, 3), (3, 2), (2, 4), (4, 2), (3, 4), (4, 3)\}$  : **FALSE**

7. Which of the following are possible value for $\vert A\times B\vert$, where $A$ and $B$ are sets:

8. $U = \{1, 2, 3, 4, 5\}$ is the universe, $A = \{ 2, 4 \}, B = \{ 2, 3 \}, C = \{ 2, 3, 5 \}$. Evaluate $\vert P(C)\vert *\vert A\times B\vert$.
>$\vert P(C)\vert = 2^3 = 8$
>$\vert A \times B\vert  = 2 * 2 = 4$
>$8 * 4 = 32$

12. $U = \{1, 2, 3, 4, 5\}$ is the universe, $A = \{ 2, 4 \}, B = \{ 2, 3 \}, C = \{ 2, 3, 5 \}$. Evaluate $\vert P(B\times C)\vert$.
>$\vert B\times C\vert = \vert B\vert\vert C\vert = (2)(3)$
>$= 6\rightarrow\vert P(B\times C)\vert = 2^{\vert B \times C\vert} = 2^6$
>$= 64$

31. Let $A$ and $B$ be sets with the following cardinalities. What is $\vert A\cup B\vert$?

$$
\vert U\vert = 54 \\
\vert A\vert = 16 \\
\vert B\vert = 23 \\
\vert \overline{(A∩B)}\vert = 42 \\
$$

> $\vert \overline{(A∩B)}\vert = \vert\overline{A}\cup\overline{B}\vert = 42$
> $\vert A \cup B\vert = \vert A\vert + \vert B\vert - (\vert U\vert - \vert\overline{A}\cup\overline{B}\vert)$
> $= 16 + 23 - (54 - 42)$
>$=27$
